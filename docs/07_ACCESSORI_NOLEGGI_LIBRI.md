# 07 - Accessori / noleggi / libri (AS-IS da ODS/Excel)

Fonte principale: `docs/materiale cliente/Db Accessori 2025-26.ods` → foglio `accessori`.

## Cosa permette di fare (oggi, in Excel/ODS)

- gestire accessori per studente (più righe + dettagli)
- gestire noleggio strumenti (cauzione, mesi, valore mensile, scadenza)
- tracciare acquisto strumento (campo + dettagli)
- tracciare libri venduti/associati (liste)
- calcolare totali accessori

## Dati/colonne rilevanti (estratto)

Da `accessori`:
- `accessori`, `tot accessori`
- `cauzione noleggio`, `dettagli noleggio`, `Mesi noleggio`, `valore noleggio mensile`, `Data scadenza noleggio`
- `Accessori 1..7` + `dettagli`
- `acquisto strumento` + `dettagli`
- `libri` + `Libri 1..7`
