# 10 - Modulo anagrafica e disponibilità (AS-IS da XLSX)

Fonte principale:
- `docs/materiale cliente/Anagrafica e disponibilità a.s. 2025_26 (Risposte).xlsx` → `Risposte del modulo 1`
- `docs/materiale cliente/Copia di Anagrafica e disponibilità a.s. 2025_26 (Risposte).xlsx` (copia)

## Cosa permette di fare (oggi, in Excel/ODS)

- raccogliere anagrafica + disponibilità tramite modulo (Google Form / equivalente)
- raccogliere scelte (laboratorio teoria, musica d’insieme/orchestra)
- raccogliere consenso trattamento dati (privacy)
- raccogliere preferenze, impegni fissi e note libere utili per costruzione orario

## Dati/colonne rilevanti (estratto)

Da `Risposte del modulo 1`:
- dati allievo (cognome/nome, data nascita, codice fiscale)
- dati genitori (cognome/nome genitore 1/2, cell, email)
- contatti allievo (cell/email se presenti)
- corso frequentato + scelte (laboratorio, orchestra)
- disponibilità per giorni (lun–ven + sabato)
- impegni fissi + preferenze + note
- consenso trattamento dati
